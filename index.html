The problem

Budgeting apps feel like work.

Many users stop tracking their money because traditional budgeting tools require too many steps: selecting categories, entering forms, checking charts, and manually understanding what went wrong.

× Manual expense entry feels slow and repetitive.
× Charts show numbers but do not explain what they mean.
× Users often notice budget problems only after they already overspent.
× Most apps track the past, but do not guide the next decision.
The solution

BudgetPal acts like a money copilot.

BudgetPal understands natural input, updates the budget, warns the user, and explains what to do next. It turns budgeting from a manual task into a guided conversation.

✓ Users can type simple inputs like coffee 40.
✓ The agent categorizes expenses and updates the right budget.
✓ Smart warnings appear before the user loses control.
✓ Users can ask questions and get personalized answers.
What BudgetPal does

A complete budgeting experience built around an AI agent.

BudgetPal combines budget tracking, natural transaction input, AI insights, spending analysis, warnings, and financial guidance into one simple mobile experience.

₪

Safe-to-spend view

Instead of only showing a balance, BudgetPal tells the user how much they can safely spend while considering budgets, goals, income, and spending pace.

AI

Natural transaction input

Users can type messy inputs like “coffee 40” or “netflix 39.90”. The agent understands the meaning and turns it into a structured transaction.

↳

Smart categorization

The agent matches expenses to existing categories, suggests subcategories, and can create new categories when needed without making the budget messy.

!

Budget warnings

When a category is close to its limit or over budget, BudgetPal explains the issue clearly and suggests a realistic next step.

?

Ask about spending

Users can ask questions like “What did I spend most on this month?” and get answers based on real transaction data.

★

Personalized saving tips

BudgetPal gives advice based on the user’s actual budget behavior, not generic tips that ignore their situation.

The agent is the product

Not just a chatbot. Not just a tracker.

BudgetPal’s agent can talk normally, perform budget actions, answer spending questions, explain trends, and prepare changes for the user to approve. It is designed to be helpful even when the user does not know exactly what to ask.

✓ Casual: “Hi” → friendly greeting using the user’s name.
✓ Action: “coffee 40” → add an expense and update the budget.
✓ Analysis: “What did I spend most on?” → category breakdown.
✓ Guidance: “How can I save?” → personalized recommendations.

How it works

From messy input to budget clarity.

BudgetPal turns natural language into safe, validated, useful financial actions.

1

User speaks naturally

The user types something simple like “coffee 40” or asks a budget question.

2

Agent understands intent

BudgetPal detects whether this is a transaction, question, advice request, or action.

3

Budget context loads

The app checks categories, limits, recent transactions, and user preferences.

4

Action is validated

The backend validates the result before saving anything to the budget.

5

User gets clarity

The budget updates, and the user receives a clear explanation or warning.
